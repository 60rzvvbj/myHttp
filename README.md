## 运行方式

### 初始化

- 执行 `npm i`

### server

- 在 `src/server` 写代码，在 `src/test/server.js` 测试调用
- 执行 `npm run server`

### client

- 在 `src/client` 写代码，在 `src/test/client.js` 测试调用
- 执行 `npm run client`
